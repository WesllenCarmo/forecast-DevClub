
# Forecast - DevClub

The project is a single-page forecast application built with <a href="https://openweathermap.org/">OpenWeather API</a>.
    
This project was made in the video lesson of how to create a foreact app in React. <a href="https://www.youtube.com/watch?v=1mm_C-t_Vyg&t=1292s">Click here</a> to access the video lesson in YouTube.

## Features

- Use of the OpenWeather API to get data about cities' forecast
- Resuable component for better API data's organization

## What I learned

- How to call and get data from an API using Axios
- How to use UseState and UseRef to catch the cities' name from an input element to get its data from the API
